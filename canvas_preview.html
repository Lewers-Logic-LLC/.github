🔄 Recycler.IQ Project Overview

Project Status: Active development across 6 sub-projects with 286 total Jira issues. Sprint 1 (Mar 29 – Apr 19, 2026) is 68% complete with RCORE nearly finished at 98% completion.

📋 Project Summary

Recycler.IQ is a comprehensive cash management platform designed to integrate with hardware cash recyclers, specifically targeting the ATEC LT380 and Kisan KR10 devices. The project encompasses both client applications and cloud infrastructure to provide real-time cash inventory tracking, predictive analytics, and automated logistics management for financial institutions.

🏗️ Architecture & Technology Stack

  • .NET MAUI Blazor Hybrid for cross-platform client applications
  • Blazor Server for web-based interfaces and simulator components
  • ML.NET for predictive cash inventory analytics and forecasting
  • Hardware Integration via native SDKs for cash recycler communication

Microservices architecture with independent CI/CD pipelines per sub-project.
