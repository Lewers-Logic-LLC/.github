# Recycler.IQ

**Status:** Sprint 1 in progress (Mar 29 — Apr 19, 2026)  
**Target GA:** July 31, 2026

## Overview
ML-powered cash flow optimization product for recyclers. Primary focus on tax authority market, secondary on retail (c-stores, cannabis).

## Sub-Projects
- **RCORE** — Shared domain & packages (substantially complete)
- **RCLIENT** — MAUI Blazor + Blazor Server clients (50% complete)
- **RCLOUD** — Cloud API & portal (backlog)
- **RLINK** — Edge service & hardware abstraction (backlog)
- **RIQSIM** — Hardware simulator for dev/demo (backlog)
